# Система массового обслуживания с абсолютным приоритетом
---
Программа для имитационного моделирования системы массового обслуживания с абсолютнмы приоритетом
---
### Необходимые компоненты:
* Python >= 3.5
* PyQt >= 5.6
* NumPy >= 1.12
* Matplotlib 2.0.1

---
### Запуск
``` cmd
python main.py
```
